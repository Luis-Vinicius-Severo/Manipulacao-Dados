# 📊 Manipulação de Dados

Projeto desenvolvido durante meus estudos de **TypeScript**, com o objetivo de aplicar na prática os principais recursos da linguagem.

A aplicação consome uma API de transações, normaliza os dados recebidos e exibe as informações em uma tabela, além de gerar estatísticas como o valor total das vendas, quantidade de transações, formas de pagamento, status das transações e o dia da semana com maior número de vendas.

## 📚 Conceitos praticados

- Consumo de API com `fetch`
- Interfaces
- Generics
- Modularização do código
- Conversão de datas
- Conversão de valores monetários
- Manipulação de arrays (`map`, `reduce` e `forEach`)
- Cálculo de estatísticas a partir dos dados

## ⚙️ Como executar

Clone o repositório:

```bash
git clone https://github.com/Luis-Vinicius-Severo/Manipulacao-Dados.git
```

Acesse a pasta do projeto:

```bash
cd Manipulacao-Dados
```

Instale as dependências:

```bash
npm install
```

Execute o projeto:

```bash
npm run dev
```
